Plataforma de Gestión para Salones de Belleza

¿Qué es este sistema?
Es una aplicación web que te permite gestionar todos los aspectos de tu salón de belleza: clientes, servicios, profesionales, recursos físicos y turnos.

Acceso a la Aplicación

Para acceder al sistema, necesitas un navegador web (Chrome, Firefox, Edge o Safari) y conexión a internet.

URL de la aplicación: https://peluqueria-lac.vercel.app/

Primera vez: Al entrar por primera vez, verás el Dashboard con estadísticas en cero. ¡No te preocupes! A medida que cargues datos, las métricas se actualizarán automáticamente.

Interface Principal

La aplicación tiene un menú lateral izquierdo con las siguientes secciones:

  • Dashboard: Resumen y estadísticas del día
  • Clientes: Gestión de clientes registrados
  • Servicios: Catálogo de servicios ofrecidos
  • Profesionales: Empleados del salón
  • Recursos: Sillas, camillas, cabinas
  • Turnos: Reserva y gestión de turnos
  • Ventas: Historial de servicios completados

En la parte superior derecha encuentras el botón para cambiar entre modo claro y modo oscuro (ideal para trabajar de noche).

Primeros Pasos: Configuración

Antes de comenzar a atender clientes, necesitas configurar los datos básicos del salón. Sigue este orden:

1. Agregar Profesionales

Ve a la sección Profesionales y agrega a cada empleado:

  • Nombre del profesional
  • Especialidad (ej: coloración, corte, manicura)
  • Estado: Activo/Inactivo
Consejo: Agrega primero a todos tus profesionales, luego será más fácil crear los turnos.

2. Agregar Recursos

Ve a la sección Recursos y registra los espacios físicos:

Tipo Ejemplo Descripción
Silla Silla 1, Silla 2 Para cortes y peinados
Camilla Camilla 1 Para masajes
Cabina Cabina color Para aplicación de tintura
Estación Estación uñas Para manicura/pedicura

3. Agregar Servicios

Ve a la sección Servicios y crea cada servicio:

  • Nombre: Ej: "Corte de cabello"
  • Categoría: haircut, color, makeup, nails, massage, other
  • Duración: Tiempo en minutos (ej: 30, 45, 60)
  • Precio: Valor del servicio
  • Tipo de recurso: Qué recurso necesita (silla, camilla, cabina)
Importante: Define bien la duración de cada servicio. El sistema usa este dato para calcular los horarios disponibles automáticamente.

4. Agregar Clientes

Ve a la sección Clientes y registra a tus clientes:

  • Nombre completo
  • Teléfono (importante para comunicaciones)
  • Email (opcional)
  • Notas (alergias, preferencias, etc.)

Cómo Reservar un Turno

Una vez configurado todo, puedes comenzar a reservar turnos.

1. Crear un Nuevo Turno

Ve a Turnos y sigue estos pasos:

  1. Haz clic en "Nuevo Turno"
  2. Selecciona el cliente (o crea uno nuevo)
  3. Elige el servicio que necesita
  4. Elige el profesional de preferencia
  5. Selecciona la fecha y hora
  6. El sistema verificará automáticamente que el profesional y el recurso estén disponibles
  7. Agrega notas si es necesario
  8. Haz clic en "Guardar"
El sistema hace todo: No necesitas calcular horarios. El sistema verifica disponibilidad del profesional, del recurso físico (silla/camilla) y del tiempo necesario para el servicio automáticamente.

2. Estados de un Turno

Estado Significado Qué hacer
Pendiente Turno reservado, sin iniciar El cliente aún no ha llegado
En curso El servicio está realizándose El profesional comenzó el trabajo
Completado El servicio finalizo Se registró la venta
Cancelado El turno fue cancelado No se atendió

3. Atender un Turno

Cuando el cliente llega y comienza el servicio:

  1. Busca el turno en la lista
  2. Haz clic en "Iniciar" para marcar como "En curso"
  3. Cuando finishes, haz clic en "Completar"

Al completar, el sistema automáticamente guarda el turno en Ventas con el precio del servicio.

Cómo Usar el Dashboard

El Dashboard es tu pantalla principal. Muestra:

  • Clientes: Total de clientes registrados
  • Turnos: Turnos del día (pendientes + en curso)
  • Servicios: Cantidad de servicios en el catálogo
  • Ventas: Total de ventas del día

Ver Turnos de Otros Días

Por defecto, el Dashboard muestra los turnos del día de hoy. Para ver otros días:

  1. Haz clic en la tarjeta de Turnos
  2. Usa el selector de fecha para elegir otro día
  3. La lista se actualiza automáticamente

Ver Detalle de un Turno

En la lista de turnos del día, haz clic en cualquier turno para ver:

  • Datos completos del cliente
  • Profesional asignado
  • Recurso asignado
  • Horario completo
  • Precio del servicio
  • Notas

Consejos Prácticos

Para aprovechar mejor el sistema:

  • Registra todos los clientes: Así evitas tomar turnos sin tener datos de contacto.
  • Mantén actualizado el catálogo: Agrega nuevos servicios cuando cambie tu oferta.
  • Usa las notas: Anota preferencias, alergias o fórmulas de color.
  • Modos oscuro: Úsalo de noche para no cansar la vista.
  • Revisa el Dashboard cada mañana: Así ves los turnos del día.
Modo oscuro: Haz clic en el ícono de sol/luna cerca del menú para cambiar entre modo claro y oscuro.

Problemas Comunes

Problema Causa probable Solución
No puedo crear un turno Horario ocupado Elige otro horario o profesional
No aparece el recurso No fue creado Ve a Recursos y agrégalo
No carga la página Sin internet Verifica tu conexión
El precio aparece en 0 No se definió precio Edita el servicio y agrega precio
Soporte: Si tienes problemas técnicos que no puedes resolver, contacta al administrador del sistema.

Seguridad y Respaldo

  • No compartas tu URL con personas de confianza
  • Haz respaldos: Exporta tus datos periódicamente (consulta al administrador)
  • Contraseñas: El sistema no maneja contraseñas de usuario por ahora

Resumen de Funcionalidades

Módulo Acciones disponibles
Dashboard Ver estadísticas, turnos del día, seleccionar fecha
Clientes Crear, editar, eliminar, buscar
Servicios Crear, editar, listar por categoría
Profesionales Crear, editar, activar/desactivar
Recursos Crear, editar, asignar tipo
Turnos Crear, iniciar, completar, cancelar
Ventas Ver historial, filtrar por fecha, servicio y/o profesional

Atajos del Teclado

  • Ctrl + S: Guardar cambios en formularios
  • Esc: Cerrar ventanas modales
  • F5: Actualizar la página
¡Listo para usar! Con estos conocimientos básicos, ya puedes comenzar a gestionar tu salón de manera profesional.
